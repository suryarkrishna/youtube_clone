Regular Expressions

regex101.com can be used to check our Regular Expressions
  1. /pattern-string/flag

    pattern-string is the string to be matched. flag indicates the additional conditional of global or insensitive.

    eg: /pen/gi

    this matches the string pen globally i.e, matches all occurences both capital and small case.

  2. Sets & Ranges

    Sets matches any of the values where as Range matches the range of values. Both are indicated using Square brackets.

    eg: Set- /[ABC]ir/gi ; Range- /[A-Z]ir/gi

    Set matches for Air, Bir, Cir where as Range matches for Air, Bir, Cir.........,Zir

    Range of 0 - 9 is written as /[0-9]/ or /[\d]/

    /[^0- 9]/ or /[\D]/- not numbers are selected

    /[a-zA-Z]/ - all alphabets both capital and small case

    /[a-zA-Z0-9]/ or /[\w]/ - selects all alphabets and numbers

    /[\W]/ - selects non-alphabets and non-numbers

    /[\s]/ - selects spaces

  3. Quantifiers

    Quantifiers select a value multiple times. To define a quantifierm we add a value inside a curly braces after the set or range.

    eg: /[\d]{3}/

    selects number with three digits

    Range can also be defined for the quantifiers- /[\d]{2,4}/ - selects numbers with 2 , 3 and 4 digits.

    /[\d]{1,}/ or /[\d]+/ - selects one or more digits

    Regex for PAN- /^[a-zA-z]{5}[0-9]{4}[a-zA-Z]{1}$/

    ? -> {0,1} i.e, 0 or 1 occurences; * -> {0,} i.e, 0 or more occurences

  4. Regex Groups
    • eg: phone number: /^[\d]{10}$/g
    • eg: phone number with +91 : /^(\+91)?[\d]{10}$/g
    • eg: url : /^www\.[\w]+\.(com|co|in|org)$/g

Regular Expressions in Js

We can use /pattern-string/flag or Inbuild Regular Expression class- new RegExp('pattern-string','flag')
